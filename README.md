## 2 需求分析与设计     
### 2.1 需求分析     
班级学委在收集班级作业时，常会面临以下问题：     
- 所交的作业名称格式有误（比如：学号+姓名的命名模式）；     
- 所交的作业格式有误（比如：PDF，DOC等等）；     
- 查询未提交作业的同学名单时比较麻烦；     
- 当同一时间段收集不同科目作业时，难以快速地分类打包。     
因此，我们决定设计一款作业提交系统来帮助班级学委更方便地收集作业，班级同学也可以方便地提交自己的作业。
